# Blox frontend challenge

### Este projeto foi criado através do "create-react-app" com template typescript e com a flag --use-npm para que o mesmo desse preferência ao npm e não ao yarn como ocorre por padrão.

## Progresso do projeto: Este repositório possui neste momento apenas a base de sua estrutura total.

## Tecnologias do projeto:

        - Typescript
        - React
        - Material-ui
        - Styled components
        - React-router-dom
        - React-use
        - Axios
        - Context-api
        - Web-hooks
        - phosphor-icons
